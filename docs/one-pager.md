# SOC AI Agent Demo One-Pager

## What It Shows

A one-click SOC incident investigation pipeline with visible LangGraph concepts: supervisor routing, parallel tool fan-out, checkpointing, human approval, time-travel replay, and final incident reporting.

## Why It Matters

SOC teams need automation that is auditable, interruptible, and explainable. This demo makes every model call, tool request, checkpoint, and analyst decision visible.

## Stack

React, TypeScript, Vite, Netlify Functions, Server-Sent Events, Z.ai GLM-5.1, GLM-5-Turbo, Playwright, Vitest.

## Proof Points

- Ten enterprise tools called before approval.
- Fourteen GLM-backed tool endpoints implemented.
- Live approval interrupt and resume.
- Checkpoint fork/replay.
- Downloadable JSON report and browser-native print/save-to-PDF report.
- Full local verify and browser smoke passing.
