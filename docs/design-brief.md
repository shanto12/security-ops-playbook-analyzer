# Design Brief

Audience: SOC automation, security AI, and LangGraph interview panels.

First 30 seconds should prove:

- One-click automated incident investigation.
- Visible graph mechanics rather than a generic dashboard.
- Real enterprise tool fan-out with transparent API logs.
- Human approval before containment.
- Checkpoint replay/fork controls.

Visual direction:

- Minimal futuristic SOC console.
- Dark background, restrained cyan/green/red semantic accents.
- Dense but scannable operational layout.
- No marketing hero, no decorative filler, no company branding.

Primary workflow:

Generate Incident -> Watch graph -> Inspect logs/checkpoints -> Approve containment -> Export report -> Fork checkpoint.
