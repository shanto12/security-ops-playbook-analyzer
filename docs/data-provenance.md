# Data Provenance

- Incidents are synthetic and generated live by Z.ai GLM-5.1.
- Enterprise tool responses are synthetic and generated live by Z.ai from the current incident context.
- No real customer data, employee records, tickets, logs, endpoints, or third-party telemetry are used.
- Screenshots are local captures of the built demo UI.
- The app is an independent concept demo and is not affiliated with any vendor whose tool category is represented.
